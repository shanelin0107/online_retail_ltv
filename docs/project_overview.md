# Project Overview

## Project Title
**Online Retail Customer Lifetime Value (LTV) Analysis**

## Executive Context
This project converts transaction-level retail activity into a decision-ready customer value model for business stakeholders. The main outcome is a Tableau dashboard that identifies value concentration, repeat behavior impact, and retention/upsell opportunities.

## Business Objectives
- Quantify the overall customer value profile.
- Compare repeat-customer value against the full customer base.
- Assess revenue concentration among high-value customers.
- Diagnose whether high value is frequency-driven, AOV-driven, or mixed.
- Recommend practical growth actions for retention and VIP strategy.

## Analytical Scope
- **Time filter in dashboard:** Invoice years 2010 and 2011.
- **Granularity:** Customer level.
- **Core dimensions:** Customer ID, Invoice Year.
- **Core measures:** Revenue, LTV, Order Frequency, AOV, Repeat Flag.

## Deliverables
1. Customer-level analytical dataset.
2. Tableau dashboard with KPI cards and behavior visuals.
3. Business recommendations tied to measurable actions.
4. Portfolio-ready repository with transparent methodology.
