# Online Retail Customer Lifetime Value (LTV) Analysis

> **Portfolio Summary:** Built an end-to-end customer LTV analytics solution that translates transaction-level online retail data into executive KPIs, behavioral customer segmentation, and retention-focused business actions in Tableau.

## 1) Business Problem
Online retail businesses often monitor total sales but lack a clear view of **who drives long-term value** and **which customer behaviors should be prioritized**. Without customer-level LTV visibility, retention and upsell investments can be misallocated.

## 2) Project Objective
This project develops a customer-level LTV analytical layer and Tableau dashboard to answer five core questions:

1. What is the overall customer value profile?
2. Do repeat customers generate more value than the average customer?
3. Is revenue concentrated among a small number of high-value customers?
4. Is value driven more by purchase frequency or by average order value?
5. Which actions can improve retention, expand customer value, and protect VIP customers?

## 3) Dataset Overview
**Source data:** Transaction-level online retail records.

**Expected raw columns:**
- `Customer ID`
- `Invoice No` / `Order ID`
- `Invoice Date`
- `Quantity`
- `Unit Price`

**Primary analytical grain:**
- Customer level (one row per customer for dashboard modeling)

## 4) Data Preparation Summary
The pipeline follows a business-first transformation approach:

1. **Data cleaning**
   - Exclude rows with missing `Customer ID` (required for customer-level LTV).
   - Flag or remove abnormal records (negative quantity, zero/negative price, canceled invoices).
   - Standardize invoice date formats and derive invoice year.
2. **Revenue creation**
   - Compute transaction revenue: `Revenue = Quantity * Unit Price`.
   - Aggregate revenue to customer level.
3. **Behavioral metrics**
   - Calculate order frequency using distinct invoice count.
   - Derive AOV from customer revenue and order frequency.
   - Build repeat flag (`Order Frequency > 1`).
4. **Analytical fields**
   - Create LTV bins for distribution views.
   - Build KPIs such as Repeat Customer %, Average LTV, and Average Repeat LTV.

## 5) KPI & Metric Definitions
| Metric | Definition | Business Interpretation |
|---|---|---|
| **Average LTV** | `AVG(Customer Total Revenue)` | Typical customer value across the population. |
| **Total Customers** | `COUNTD(Customer ID)` | Active known customer base size. |
| **Average Repeat LTV** | `AVG(Customer Total Revenue where Repeat Flag = 1)` | Average value generated by repeat customers. |
| **Repeat Customer %** | `Repeat Customers / Total Customers` | Customer retention quality indicator. |
| **Order Frequency** | `COUNTD(Invoice No)` by customer | Purchase cadence. |
| **AOV** | `Customer Total Revenue / Order Frequency` | Basket value proxy. |

## 6) Dashboard Overview
The Tableau dashboard includes:

1. **KPI Cards**
   - Average LTV
   - Total Customers
   - Average Repeat LTV
   - Repeat Customer %
2. **Top 20 Customers by LTV**
   - Horizontal bar chart of highest-value customers.
3. **Order Frequency vs AOV**
   - Scatter plot to diagnose if high value comes from frequency, AOV, or both.
   - Annotated behavioral groups: high-value, high-AOV, frequent customers.
4. **LTV Distribution**
   - Binned/histogram-style view of customer value concentration.
5. **Invoice Year Filter**
   - Time slicing for 2010 and 2011.

## 7) Key Insights
1. Repeat customers generate higher value than the overall customer base.
2. Revenue is concentrated among a relatively small segment of high-value customers.
3. LTV distribution is right-skewed: many low-to-mid value customers, with a long high-value tail.
4. High LTV is behaviorally diverse:
   - some customers purchase frequently,
   - others purchase less often but at higher AOV.
5. The strongest near-term growth levers are retention and VIP protection.

## 8) Business Recommendations
1. **Prioritize repeat-customer retention** through loyalty design, personalized offers, and re-engagement journeys.
2. **Protect and grow VIP customers** via exclusive benefits, service differentiation, and account-level CRM playbooks.
3. **Upsell frequent low-AOV customers** using bundles, cross-sell paths, and premium product recommendations.
4. **Re-activate low-frequency customers** with win-back campaigns and first-repeat incentives.
5. **Expand next dashboard iteration** with Pareto analysis, top 10% contribution tracking, and deeper segmentation lenses.

## 9) How to Reproduce
1. Place raw data file(s) into `data/raw/`.
2. Run the SQL starter logic in `sql/customer_ltv_analysis.sql` to build cleaned and customer-level tables.
3. Optionally run `scripts/data_preparation.py` for a Python-based preprocessing workflow.
4. Export customer-level analytical dataset to `data/processed/customer_ltv_dataset.csv`.
5. Connect Tableau workbook (`dashboards/tableau/online_retail_ltv_dashboard.twbx`) to the processed dataset.
6. Build dashboard components and publish screenshots to `dashboards/screenshots/`.
